Question 2 : Longer the arrow, Easier it is to hit

class Arrow{
  Log log := Log.get("arrow")
  
  Void main(){
    toStr := 3
    echo(toStr.toStr)
    echo(toStr -> toStr)
    echo(toStr --> toStr)
  }
}
→
Will run and print 
3
3
3
Will run and print something else
Will not compile because toStr is not a valid identifier
Will not compile because ---> is not a valid operator

Question 3 : The Nasty Actor

using concurrent

class LogTheError{
  Log log := Log.get("something")

  Void main(){
    a := Actor(ActorPool())|msg|{
        try{
           log.info("called with $msg")
           return msg.toStr
        }
        catch(Err e){
            log.err("something bad", e)
            return "error"
        }
    }
    
    log.info(a.send(2).get)
  }
}
→

called with 2
